= データセットを表示する
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
データセットに関連付けられている仮想マシンを表示できます。

. ナビゲーションペインで、* Protection *>* Datasets *をクリックします。
. ツリービューを展開して、データセットに属する仮想マシンを確認します。
