= ポリシーを表示します
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ./media/


[role="lead"]
特定のデータセットに関連付けられているポリシーの詳細を表示できます。

.手順
. ナビゲーションペインで、［メニュー］、［保護］、［データセット］の順にクリックします。
. 表示するポリシーが含まれているデータセットを選択します。
. [Policies]ペインで、詳細を表示する特定のポリシーを選択します。
+
ポリシーに関する情報が[Details]ペインに表示されます。
