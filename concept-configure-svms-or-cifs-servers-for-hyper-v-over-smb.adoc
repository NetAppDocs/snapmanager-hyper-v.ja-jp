= Hyper-V over SMB用のSVMまたはCIFSサーバを設定
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
1台のStorage Virtual Machine（SVM）またはCIFSサーバを複数のアプリケーション用に設定すると、リソース共有の問題が発生し、その結果Hyper-V環境に影響が及ぶ可能性があります。要件に応じて、Hyper-V over SMB専用のSVMまたはCIFSサーバを設定する必要があります。
