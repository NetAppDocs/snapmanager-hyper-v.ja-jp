= Eメール通知の設定
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
通知の複数のEメール受信者は、カンマで区切る必要があります。

SnapManager for Hyper-VでEメール通知の受信者を複数設定する場合は、各受信者をカンマで区切ります。この要件は、SnapManager for SQLとは異なります。SQLでは、各Eメール通知の受信者をセミコロンで区切る必要があります。
